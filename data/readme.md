README_sources

Dataset scope
-------------
This dataset standardizes approximate metro route/network length values (km) for 15 global cities at 7 time steps: 1990, 1995, 2000, 2005, 2010, 2015, and 2020.

Important note on “correctness”
-------------------------------
Metro systems are reported differently by different institutions. Depending on the source, values may refer to:
- route length
- network length
- track length

For this final lab revision, the dataset was normalized to route/network length where possible. When official operator websites did not provide a year-by-year historical table, values were compiled from official system-history pages and well-documented system overviews listing line lengths and opening dates.

This is therefore a standardized historical dataset for comparative mapping, not an engineering audit table.

Cities corrected most substantially after instructor feedback
-------------------------------------------------------------
- London: 1990 corrected from 0 to 402
- Moscow: 1990 corrected from 0 to 239
- Beijing: early network values corrected to reflect the pre-2000 system
- Cairo: early network values corrected to reflect operation after 1987
- Istanbul: early network values corrected to reflect the first metro line
- Bangkok: early values corrected to reflect system opening around 1999–2000

Notes
-----
Metro system length definitions can vary slightly between sources (route length vs track length vs network length). For this lab, values represent approximate route/network length and were selected to illustrate broad temporal expansion patterns.
